## Installation

Install the dependencies and devDependencies and run the app.

```sh
yarn install
```

## Run IOS

```sh
yarn ios
```

## Run Android

add local.properties file in ./android folder  and run command:
```sh
yarn android
```

## Hygen Commands

```sh
yarn g:screen
yarn g:screen-component
yarn g:component
```
